What is a box model in CSS?

The CSS box model describes the space that surrounds each HTML element. It consists of content, padding, border, and margin. The box-sizing property defines how the width and height of an element are calculated, taking into account the padding and border, or just the content area.

What is a “clearfix hack” and when does it need to be applied?

A clearfix hack is a technique used to contain floats within a container. It's applied when a parent element does not fully expand to contain its floated children. This typically happens when the children are floated and the parent has no height or collapsed height.

What is a “margin collapse” and what are the use cases for margin: auto and negative margin values?

Margin collapse occurs when the vertical margins of two adjacent elements overlap, collapsing into a single margin. This commonly happens between siblings or a parent and its first/last child. The margin: auto; property is used to horizontally center an element within its container, while negative margin values can be used to create overlapping or visually interesting layouts.

What was the problem with achieving Holy Grail layout?

The Holy Grail layout is a classic web design pattern that consists of a header, footer, and three columns of content, with the center column being the main content and the side columns being of equal height. The problem with achieving this layout traditionally was making all columns equal height without resorting to non-semantic markup or JavaScript hacks.

What are the modern ways to achieve Holy Grail layout?

Modern ways to achieve the Holy Grail layout include using Flexbox or CSS Grid layout. Flexbox allows for easy alignment and distribution of elements along a single axis or multiple axes, while CSS Grid provides a powerful grid-based layout system for more complex designs.
